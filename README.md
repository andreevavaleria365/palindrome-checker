# Лабораторная работа №5

## Проверка палиндрома

![Tests](https://github.com/andreevavaleria365/lab5-palindrom/actions/workflows/test.yml/badge.svg)

### Описание

Программа проверяет, является ли строка палиндромом. 
Палиндром — это слово, фраза или число, которые читаются одинаково слева направо и справа налево.

### Особенности

- Игнорирует пробелы
- Игнорирует регистр букв (заглавные и строчные считаются одинаковыми)
- Работает с русскими и английскими буквами
- Работает с цифрами

### Примеры работы

| Входная строка | Результат |
|:--|:--|
| `А роза упала на лапу Азора` | ✅ палиндром |
| `racecar` | ✅ палиндром |
| `hello` | ❌ не палиндром |
| `12321` | ✅ палиндром |
